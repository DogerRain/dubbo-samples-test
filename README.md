## 1、介绍

本项目是比较 RPC框架 Dubbo 和 Thrift 的差别，以及两者的使用。

学习本项目你可以收获：

- dubbo的使用
- Thrift的使用
- 压测工具jmeter、ab（Apache-BenchMark）、jmh 的使用
- dubbo、thrift的性能
- log4j2（漏洞已升级）、logback的使用
